# 📚 Virtual Study Room

A distraction-free online study space designed to help students focus, manage time, and improve productivity.

## 🚀 Live Demo
👉  https://vijuniya-creator.github.io/virtual-study-room/


## ✨ Features

- ⏱️ Timer for focused study sessions  
- 📋 To-Do List for task management   
- 🔔 Break reminders for better productivity  
- 🧘 Minimal and distraction-free UI  


## 🛠️ Built With

- HTML  
- CSS  
- JavaScript  


## 📌 How to Use

1. Open the website  
2. Add your tasks  
3. Start the study timer  
4. Take breaks when prompted  
5. Stay focused and productive 🚀  



## 💡 Future Improvements

- Dark/Light mode toggle  
- Daily progress tracking  
- Login system
